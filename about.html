About & References

This site is an HTML/CSS project that explains philosophical ideas expressed through games. It avoids libraries and demonstrates unique HTML features via clear, commented examples.

Unique Elements

  • <ruby> – annotate terms (e.g., Japanese) with pronunciation.
  • <cite> – mark titles of works (games, books, articles).
  • <noscript> – provide fallback content if JavaScript is disabled.

MDN

  • MDN: <ruby>
  • MDN: <cite>
  • MDN: <noscript>

Unique Attributes

  • ismap (on <img>) – enables server‑side image mapping when the image is inside a link.
  • ping (on <a>) – specify endpoints to notify (POST) when the link is followed.
  • formaction (on submit controls) – override a form’s action per button.
